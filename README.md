🌸 Iris Flower Classification
A Machine Learning project to classify iris flowers into different species with high accuracy.

📌 Overview
This project uses Machine Learning algorithms to classify iris flowers into three species:

🌼 Setosa
🌿 Versicolor
🌺 Virginica

Based on flower measurements, the model predicts the correct category.

🎯 Project Highlights

✨ Clean and well-structured dataset
✨ Multiple ML models applied
✨ Data visualization included
✨ High accuracy achieved
✨ Beginner-friendly ML project

📊 Dataset Features
Feature Name	Description
Sepal Length	Length of sepal (cm)
Sepal Width	Width of sepal (cm)
Petal Length	Length of petal (cm)
Petal Width	Width of petal (cm)
